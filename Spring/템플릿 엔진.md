
# 템플릿 엔진

### 템플릿 엔진이란
**지정된 템플릿 양식과 데이터**가 합쳐져 HTML문서를 출력하는 소프트웨어를 의미

#### 서버 템플릿 엔진 (JSP, Freemarker)
서버에서 Java 코드로 문자열을 만든 뒤 이 문자열을 HTML로 변환하여 브라우저로 전달

#### 클라이언트 템플릿 엔진 (React, Vue)
자바스크립트는 브라우저 위에서 작동함.
자바와는 다르게 서버에서 실행하지 않고 브라우저에서 실행!
